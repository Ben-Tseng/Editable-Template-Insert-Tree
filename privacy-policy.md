Privacy Policy for Editable Template Insert Tree

Editable Template Insert Tree does not collect, store, transmit, or share any personal data.

Data Collection

This extension does not collect any user information.
All templates and settings are stored locally on the user's device using the browser's local storage.

Data Usage

The extension only accesses webpage content for the purpose of inserting user-selected templates into focused input fields.

Third-Party Services

This extension does not use any third-party analytics, tracking, or external services.

Data Transmission

No data is transmitted to any external servers.

Contact

If you have any questions about this privacy policy, please contact the developer via the GitHub repository.
